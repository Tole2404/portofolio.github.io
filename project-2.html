SMART CHICKEN COOP

This page contains the case study of the Smart chicken Coop Open-Source Project, which includes the Project Overview, Tools Used, and Live Links to the project.

Project Link

Project Overview

The project I am working on is a Smart Kandang Atam using an Arduino microcontroller. This project is part of the Pemrogrammman Sistem Kendali course assignment that I am taking.

TOOL DESCRIPTION
This tool or device is an automated chicken coop, with systems in place for feeding, watering, and stabilizing the temperature within the coop. This chicken coop utilizes several sensors that operate automatically and consistently.

METHOD
The prototyping method with an embedded system pattern is automated. It uses parameters for feeding, watering, and lighting.

The hardware used includes the Arduino Uno microcontroller, DHT11 sensor, Relay, Jumper Cables, Water Sensor, Servo, RTC, Push Button, Water Pump, and Power Adapter. The software used for programming is the Arduino IDE.

This project has received recognition and was exhibited at the FTI FAIR event on my campus.
